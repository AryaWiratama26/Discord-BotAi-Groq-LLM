# Discord-bot-AI-Groq-Llama

## Jangan taruh token didalam file code, contohnya disini adalah `main.py`
![cnth-token](/ss/n-bot.png)
## - Pertama : Kamu bisa taruh token di dalam file `.env`
![env](/ss/workspace.png)
## - Kedua : Kamu bisa load token tersebut dengan cara : 
![load](/ss/load_token.png)

## Token Groq
## - Pertama : Kamu bisa dapatkan token untuk AI kamu dari website [console.groq.com](https://console.groq.com/)
## - Kedua : Kamu bisa masuk ke bagian `API Keys` untuk mendapatkan token
![groq-dash](/ss/groq-dash.png)
## - Ketiga : Klik button `Create API Key` untuk mendapatkan token 
![groq-api](/ss/groq-key.png)

## Token Discord Bot
## - Pertama : Kamu bisa masuk ke website [discord developer portal](https://discord.com/developers/docs/intro)
## - Kedua : Pada bagian `Applications`, kamu bisa klik button `New Application`:
![dc-dash](/ss/dc-dash.png)
## - Ketiga : Setelah masuk, pada bagian `Bot` kamu bisa klik `Reset Token` untuk mendapatkan token baru
![restok](/ss/res-tok.png)
